## Dokumentazioa sortu edo ikusi

DocFX instalatuta badago, dokumentazioa berriz sortzeko:

```powershell
cd GastuakApi
docfx docfx.json
```

Dokumentazioa web moduan ikusteko:

```powershell
cd GastuakApi
docfx docfx.json --serve
```

APIa exekutatzen denean Swagger dokumentazioa ere erabil daiteke:

```text
/swagger
```
